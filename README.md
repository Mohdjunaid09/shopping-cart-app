# 🛒 Shopping Cart Web Application

## 📌 Description
This project is a responsive shopping cart web application developed using HTML5, CSS3, JavaScript, and Bootstrap. It allows users to register, login, browse products, and manage their cart.

---

## 🚀 Features
- User Registration and Login
- Product Catalog
- Add to Cart functionality
- Cart Page to view selected items
- Responsive design using Bootstrap
- Data stored using LocalStorage

---

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript
- Bootstrap
